﻿# Adopt_Animal
 🐾 AdoptaPet - Application d'Adoption d'Animaux :: 

🚀 Fonctionnalités Principales :

  🏠 Écran principal avec grille 2 colonnes

  ❤️ Système de favoris avec Room

  🌐 Données en temps réel depuis The Dog API & The Cat API

  📋 Processus d'adoption stocké dans BD

  🎬 Section documentaire YouTube intégrée

🛠️ Technologie :
- Kotlin + Jetpack Compose
- MVVM Architecture  
- Retrofit (API REST) 
- Room (Base de données)
- Coroutines (Async)
- Material Design 3
- Coil (Chargement d'images)

🎯 Usage
 - Parcourir les animaux disponibles

 - Ajouter aux favoris avec un clic

 - Consulter les détails complets

 - Adopter un animal

 - Regarder le documentaire dans "À Propos"
